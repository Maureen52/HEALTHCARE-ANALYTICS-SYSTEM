Healthcare Analytics and Disease Prediction System — A full end-to-end large-scale distributed data and machine learning platform built across six milestones for SDS 2412 (Analysis of Large Datasets) at JKUAT, using the Diabetes 130-US Hospitals dataset (101,766 records) to predict 30-day diabetic patient readmission risk through a Lambda Architecture spanning Apache Spark batch processing, Kafka real-time streaming, a Gradient Boosted Trees model (AUC-ROC 0.7623), FastAPI deployment, MongoDB storage, and Evidently AI drift monitoring. The system further incorporates a Federated Learning simulation (FedAvg across 8 hospital silos, ΔAUC = −0.008) for privacy-preserving ML, an interactive Streamlit clinical dashboard, and achieves a 7.1× pipeline speedup — demonstrating a complete journey from raw CSV data to a monitored, explainable, real-time AI prediction platform.
